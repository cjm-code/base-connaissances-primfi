# 📚 Base de connaissances PRIMFI

Plateforme de ressources et veille stratégique PRIMFI.

## 🚀 Fonctionnalités

- 🔍 **Recherche multi-mots** en temps réel
- 🎯 **Filtres par catégorie** et type de ressource
- 📋 **Copie du chemin** en un clic (NAS, Shadow Drive, etc.)
- 📊 **153+ ressources** organisées en 4 piliers
- 🌐 **Données en temps réel** via Google Sheets

## 🛠️ Technologie

- **Next.js 14** (React framework)
- **TypeScript** (typage statique)
- **Vercel** (déploiement)
- **Google Sheets** (données source)

## 📦 Installation locale

```bash
# Cloner le repo
git clone https://github.com/TON_USERNAME/primfi-knowledge-hub.git
cd primfi-knowledge-hub

# Installer les dépendances
npm install

# Mode développement
npm run dev

# Accès : http://localhost:3000
